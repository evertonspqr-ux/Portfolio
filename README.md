# Portfólio

Projeto em React + Vite para apresentar experiência, projetos, formação e contato.

## Rodando localmente

1. Instale as dependências com `npm install`
2. Inicie o ambiente de desenvolvimento com `npm run dev`
3. Gere a versão de produção com `npm run build`

## Estrutura

- `src/pages/Portfolio.jsx`: página principal do portfólio
- `src/lib/PageNotFound.jsx`: página 404
- `src/index.css`: tema global e variáveis de estilo
- `vite.config.js`: configuração do Vite com alias `@`

## Observações

Este projeto foi desacoplado da plataforma original e agora pode ser mantido como uma aplicação React/Vite comum.
